# Análisis del mercado de videojuegos - TripleTen - Bootcamp Análisis de Datos

El objetivo del proyecto era identificar patrones y perfiles para determinar el éxito de un videojuego. 

Para lograr esto, llevé a cabo el procesamiento de bases de datos para identificar patrones de producción, oferta y demanda de videojuegos. Asimismo, llevé a cabo la creación de perfiles por por región, pruebas de hipótesis para comparar las calificaciones a consolas y categorías de videojuegos.

Con base en los hallazgos, identifiqué la nueva consola en tendencia y los géneros de videojuegos más consumidos. Esto me permitió brindar sugerencias en las estrategias de negocio y de marketing especializado por región para garantizar el éxito de una campaña de ventas.

Herramientas utilizadas: Python
