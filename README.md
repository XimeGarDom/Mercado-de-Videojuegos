# Análisis del mercado de videojuegos
Se identificaron patrones y perfiles para determinar el éxito de un videojuego. 
Se llevaron a cabo las siguientes actividades: 
- Procesamiento de bases de datos para patrones de producción, compra y venta de videojuegos.
- Creación de perfiles de los clientes por región.
- Elaboración de visualizaciones para compartir los hallazgos
- Prueba de hipótesis para comparar las calificaciones a consolas y categorías de videojuegos.

Herramientas utilizadas: Python
